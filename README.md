# Mark — Bilingual Greek / Spanish Reader

A verse-by-verse bilingual reader for the Gospel of Mark.

- **Greek:** NA27 / UBS4 (with morphology)
- **Spanish:** Nueva Traducción Viviente (NTV, 2009)

## Layout

Each verse is stacked (Greek above Spanish) so it reads cleanly on phone without horizontal scrolling.

## Structure

```
mark-01.md   — Mark chapter 1
mark-02.md   — (to come)
...
```
